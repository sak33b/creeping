# Creeping

An interactive web narrative built for my **ENG115: Introduction to English Prose** project. Inspired by "The Yellow Wallpaper" by Charlotte Perkins Gilman, this piece explores mood, confinement, and creeping dread through visual design and branching journal entries.

## Live link

https://sak33b.github.io/creeping/

## Overview

- Multi-page, choice-driven journal entries that guide the reader through four stages.
- A visual progression from quiet unease to chaotic collapse using typography, color, and motion.
- Lightweight, static delivery with no backend.

## Tech stack

- Vite for build and dev server
- Tailwind CSS for utility styles
- Vanilla JavaScript for navigation and interactions

## Run locally

```bash
npm install
npm run dev
```
